# 🏋️ GymFlow India

A modern fitness management web application built using React, TypeScript, and Firebase.  
Designed to streamline gym operations including member tracking, workouts, bookings, and subscriptions.

---

## 🚀 Features

- 👤 User Onboarding & Profile Management  
- 📊 Dashboard with Fitness Insights  
- 💪 Workout Tracking System  
- 🧑‍🏫 Trainer Management  
- 📅 Booking System  
- 💳 Subscription Handling  
- 🏆 Leaderboard System  
- 🔥 Firebase Integration (Authentication + Database)  

---

## 🧠 Architecture

The project follows a modular React architecture:
src/
│── components/ # Reusable UI components
│── pages/ # Feature-based pages
│── services/ # API & Firebase logic
│── utils/ # Helper functions
│── firebase.ts # Firebase configuration
│── App.tsx # Main app routing

---

## 🛠️ Tech Stack

- **Frontend:** React + TypeScript  
- **Styling:** CSS  
- **Backend:** Firebase (Firestore / Auth)  
- **Build Tool:** Vite  
- **Version Control:** Git & GitHub  

---

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/damasvasree/Gym-Flow.git
cd Gym-Flow
```
### 2. Install dependencies
npm install

## 3. Configure Firebase

Create a .env file in the root folder and add:

VITE_FIREBASE_API_KEY=your_key <br>
VITE_FIREBASE_AUTH_DOMAIN=your_domain <br>
VITE_FIREBASE_PROJECT_ID=your_project_id <br>

⚠️ Without this, the project will not run.

## 4. Run the project
npm run dev
